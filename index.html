Asteroids Clone

This action packed space game is an original clone of Asteroids famous video game written in C++ and made with the SDL2 graphics library. This game captures the original game Asteroids but with a optional locked camera, and added visual effects.

Don't Lose Breathe

Don't Lose Breathe is an intense 2D top-down bullet hell game created for Bullet Hell Jam 2023. Survive relentless enemy fire through three challenging levels, culminating in an epic boss fight. Precision and quick reflexes are essential. Written in C++ and made with the SDL2 graphics library.

Slay The Creature is a thrilling Unreal Engine 5 game where players navigate a maze to find a hidden weapon. Use phone boxes to hide from a lurking monster. Stealth and strategy are crucial as you outsmart the creature to survive and slay it. Will you conquer the terror and emerge victorious?
